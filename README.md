## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run locally:**
   ```bash
   npm run dev
   ```
   Open http://localhost:3000

3. **Deploy to Vercel:**
   ```bash
   ./deploy-template.sh
   ```

## Customization
Edit `app/page.tsx` to:
- Update hero headline
- Change features
- Add your branding
- Replace placeholder content

## Tech Stack
- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS 3
- Node.js 20+

## Deployment
This template is optimized for Vercel deployment but works with any platform that supports Next.js.
